# 💌 Outstagram
***

 🙆🏻‍♀️💌🙆🏻‍♂️ Outstagram은 현재 유행하는 SNS(Instagram)의 클론코딩 입니다.  
 👫 Outstagram은 개인프로잭트로 진행되었으며, 사용한 기술은 아래와 같습니다.  

  
  

# 👩🏻‍💻 Front-end
***

## 개발언어
 JAVASCRIPT, HTML, CSS

## 기타
 BootStrap
  
  

# 👩🏻‍💻 Back-end
***

## 개발언어
 JAVA, JSP

## 프레임워크
 Spring boot(sts3)

## 데이터베이스
 MYSQL

## 기타사용
 JPA, Spring Security, AJAX
